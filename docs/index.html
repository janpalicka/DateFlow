Quick start

Date picker

Default date-only mode. Focus the field to open the calendar.

Source
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <title>Calendar picker</title>
  </head>
  <body>
    <label for="date-field">Pick a date</label>
    <input type="text" id="date-field" />

    <script type="module">
      import { createCalendarPicker } from "./path/to/calendar/index.ts";

      const input = document.getElementById("date-field");
      createCalendarPicker(input, {
        value: new Date(2026, 5, 8),
        onChange: (date) => console.log("Selected:", date),
      });
    </script>
  </body>
</html>

Getting started

Pass a text input to the picker and react to changes.

Basic picker with time

Pass your text input to createCalendarPicker. Focus it to open the calendar panel.

import { createCalendarPicker } from "@/calendar";

const input = document.getElementById("date-field");
const picker = createCalendarPicker(input, {
  value: new Date(2026, 2, 15, 12, 30),
  showTime: true,
});

Runtime setOptions

Update locale, time, or mode without destroying the picker.

const picker = createCalendarPicker(input, { onChange });

picker.setOptions({ locale: de });
picker.setOptions({ showTime: true });

picker.setOptions({
  mode: "range",
  range: { start: picker.getValue(), end: null },
});

picker.setOptions({ mode: "single", value: picker.getRange().start });

Localization

Pass a full locale object or merge partial overrides. Inspired by flatpickr localization.

German locale

German month names and Monday as the first day of the week.

import { createCalendarPicker } from "@/calendar";
import { de } from "@/calendar/locales";

createCalendarPicker(input, {
  locale: de,
  onChange: (date) => { /* … */ },
});

French locale

Another full locale with Monday as the first weekday.

import { createCalendarPicker } from "@/calendar";
import { fr } from "@/calendar/locales";

createCalendarPicker(input, {
  locale: fr,
});

Partial locale override

Merge over defaults — here Czech labels with a custom week-number header.

import { createCalendarPicker } from "@/calendar";
import { cs } from "@/calendar/locales";

createCalendarPicker(input, {
  showWeekNumbers: true,
  locale: {
    ...cs,
    weekNumberHeader: "Tý",
  },
});

Constraining dates

Limit which calendar days can be selected.

Min and max date

Only days within the inclusive range are selectable.

createCalendarPicker(input, {
  value: new Date(2026, 2, 15),
  minDate: new Date(2026, 2, 10),
  maxDate: new Date(2026, 2, 25),
  onChange: (date) => { /* … */ },
});

Disabled dates (array)

Block specific dates — useful for holidays or closures.

createCalendarPicker(input, {
  value: new Date(2026, 2, 15),
  disabledDates: [
    new Date(2026, 2, 17),
    new Date(2026, 2, 18),
    new Date(2026, 2, 19),
  ],
});

Disabled dates (predicate)

Disable weekends with a function.

createCalendarPicker(input, {
  disabledDates: (date) => {
    const day = date.getDay();
    return day === 0 || day === 6;
  },
});

Allowed dates only (array)

Whitelist mode — only listed dates are selectable.

const allowed = [5, 12, 19, 26].map(
  (day) => new Date(2026, 2, day),
);

createCalendarPicker(input, {
  value: new Date(2026, 2, 12),
  enabledDatesOnly: allowed,
});

Allowed dates only (predicate)

Only odd-numbered calendar days are selectable.

createCalendarPicker(input, {
  enabledDatesOnly: (date) => date.getDate() % 2 === 1,
});

Time selection

Optional hour and minute selectors; 12-hour mode and seconds are supported.

24-hour time

Hour 0–23 and minute in 5-minute steps (customize with minuteStep).

createCalendarPicker(input, {
  showTime: true,
  minuteStep: 1,
  value: new Date(2026, 2, 20, 14, 45),
  // default outputFormat: "yyyy-MM-dd HH:mm"
});

12-hour time

Hour 1–12 with an AM/PM selector.

createCalendarPicker(input, {
  showTime: true,
  use12HourTime: true,
  value: new Date(2026, 6, 9, 15, 30),
  // default outputFormat: "yyyy-MM-dd hh:mm a"
});

Seconds (24-hour)

showSeconds adds a seconds dropdown when showTime is enabled.

createCalendarPicker(input, {
  showTime: true,
  showSeconds: true,
  value: new Date(2026, 2, 15, 14, 30, 45),
});

Seconds (12-hour)

Hour, minute, second, and AM/PM selectors together.

createCalendarPicker(input, {
  showTime: true,
  showSeconds: true,
  use12HourTime: true,
  value: new Date(2026, 2, 15, 15, 30, 45),
});

Direct text input

allowInput lets users type values into the picker input. By default the input is read-only.

allowInput — date

Type a date using outputFormat. Blur or Enter commits; Enter closes the calendar by default.

createCalendarPicker(input, {
  allowInput: true,
  outputFormat: "yyyy-MM-dd",
  value: new Date(2026, 2, 15),
});

allowInput — 24-hour datetime

Default outputFormat includes time when showTime is true.

createCalendarPicker(input, {
  allowInput: true,
  showTime: true,
  value: new Date(2026, 2, 15, 14, 30),
  // outputFormat: "yyyy-MM-dd HH:mm"
});

allowInput — 12-hour datetime

Use use12HourTime for AM/PM parsing and display.

createCalendarPicker(input, {
  allowInput: true,
  showTime: true,
  use12HourTime: true,
  value: new Date(2026, 2, 15, 15, 30),
  // outputFormat: "yyyy-MM-dd hh:mm a"
});

Keep calendar open on Enter

keepOpenOnAllowInputEnter keeps the panel open after a successful Enter commit.

createCalendarPicker(input, {
  allowInput: true,
  keepOpenOnAllowInputEnter: true,
  showTime: true,
  value: new Date(2026, 2, 15, 14, 30),
});

Date ranges

Range mode with optional separate start/end times and custom output formatting.

Basic range

First click starts a range, second completes it. Use onRangeChange for updates.

createCalendarPicker(input, {
  mode: "range",
  range: {
    start: new Date(2026, 2, 5),
    end: new Date(2026, 2, 18),
  },
  outputFormat: "yyyy-MM-dd",
  onRangeChange: ({ start, end }) => { /* … */ },
});

Range with start & end time

Separate time controls under each calendar pane in range mode.

createCalendarPicker(input, {
  mode: "range",
  showTime: true,
  range: {
    start: new Date(2026, 3, 10, 9, 30),
    end: new Date(2026, 3, 14, 17, 45),
  },
  outputFormat: "yyyy-MM-dd HH:mm",
  onRangeChange: ({ start, end }) => { /* … */ },
});

Range separator & format

rangeOutputSeparator and outputFormat shape the value input string.

createCalendarPicker(input, {
  mode: "range",
  showTime: true,
  outputFormat: "dd.MM.yyyy HH:mm",
  rangeOutputSeparator: " → ",
  range: {
    start: new Date(2026, 8, 1, 8, 0),
    end: new Date(2026, 8, 7, 20, 0),
  },
});

Range + week numbers

Combine range mode with ISO week column.

createCalendarPicker(input, {
  mode: "range",
  showWeekNumbers: true,
  locale: { weekNumberHeader: "KW" },
  range: {
    start: new Date(2026, 0, 6),
    end: new Date(2026, 0, 20),
  },
});

Range + dark theme

theme: "dark" sets data-cal-theme on the calendar root.

createCalendarPicker(input, {
  mode: "range",
  theme: "dark",
  range: {
    start: new Date(2026, 4, 12),
    end: new Date(2026, 4, 26),
  },
});

Formatting

Control how selected values appear in the input using date-fns patterns.

Custom outputFormat

outputFormat is a date-fns pattern synced to the value input.

createCalendarPicker(input, {
  showTime: true,
  outputFormat: "EEEE, d MMMM yyyy 'at' HH:mm",
  value: new Date(2026, 7, 19, 15, 0),
});

Theming & accessibility

Built-in theme presets, CSS variables, and accessibility options.

Dark theme preset

Sets data-cal-theme="dark" and bundled CSS variables.

createCalendarPicker(input, {
  theme: "dark",
  showTime: true,
});

High-contrast preset

theme: "contrast" — strong borders and yellow accent.

createCalendarPicker(input, {
  theme: "contrast",
});

Custom CSS variables

Override --cal-* variables on the panel wrapper (getCalendarElement()) — no theme name required.

const picker = createCalendarPicker(input, { /* … */ });
const panel = picker.getCalendarElement();
panel.style.setProperty("--cal-bg", "#fff7ed");
panel.style.setProperty("--cal-accent", "#c2410c");

Forest theme + constraints

Combine theme, locale, min/max, and disabledDates.

import { de } from "@/calendar/locales";

createCalendarPicker(input, {
  theme: "forest",
  locale: de,
  minDate: new Date(2026, 2, 1),
  maxDate: new Date(2026, 3, 30),
  disabledDates: (d) => d.getDay() === 3,
  showTime: true,
});

aria-label

Custom accessible name for the calendar region.

createCalendarPicker(input, {
  ariaLabel: "Delivery date",
});

Reset button

showResetButton adds a header icon that clears the selection.

createCalendarPicker(input, {
  showResetButton: true,
  resetInputLabel: "Reset",
  value: new Date(2026, 2, 13, 14, 15),
  showTime: true,
});

Year text input

Type a year directly; values are clamped to minDate and maxDate when set.

createCalendarPicker(input, {
  minDate: new Date(2026, 2, 1),
  maxDate: new Date(2026, 8, 30),
  value: new Date(2026, 5, 1),
});
